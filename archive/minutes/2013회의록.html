삼괴지역 기독교 연합회 2012년 정기 총회 회의록

일시 : 2012년 11월 29일(목) 오전 11시

장소 : 화산교회(조시현목사 시무)

참석인원 : 10명

김충현목사, 조시현목사, 최명헌목사, 류성원목사, 엄영주목사, 박광희목사

이성진목사. 박희만목사, 박영실목사, 심상근목사, 문순애목사, 나진균목사

1부 예배 (11시 00분)

회장(엄영주목사)의 사회로 예배를 드리다.

묵도 및 사도신경 -> 찬송(597장) -> 기도 : 조시현목사 -> 설교 : 엄영주목사

설교본문 열왕기상 4 : 29~34 ‘솔로몬의지혜’ -> 광고 : 최명헌목사

축도 : 김충현목사

예배 후 바로 총회를 시작하다.

2부 회무처리 (11시 25분)

회원점명 : 서기(류성원목사)가 호명대신 참석인원수를 보고하고 그 인원이 12명이 되므로 참석인원이 곧 개회 성수이므로 회장이 개회를 선언하다.

서기보고 : 문서대로 받기로 결의하다 (동의:박영실목사, 재청:조시현목사)

총무보고 : 임원모임 9회, 월례회 1회, 정기총회 1회, 부부 야유회 2회, 집회 5회, 신년하례 회1회로 행사 및 모임을 가졌음을 보고함. (동의:박영실목사, 재청:조시현목사)

감사보고 : 감사가 출석하지 않아 회계보고만 하기로 하다.

회계보고 : 문서대로 받기로 동의하다 (동의:김충현목사, 재청:박영실목사)

임원을 선출하기 위해 박영실목사가 정회하기로 동의하다.

임원선출을 마무리하고 발표를 하다.

임원선출 회장 : 조시현목사(화산교회/합동)

상임부회장 : 최명헌목사 (염광교회/통합)

부회장 :

총무 : 이성진목사(우정교회/합동)

서기 : 류성원목사(화성권능교회/합동)

회계 : 박광희목사(조암열방순복음교회/순복음)

감사 : 엄영주목사, 김충현목사

신·구임원교체

신임회장의 인사 후 신임회장이 직전회장에게 공로패 전달 제안에 회원들의 허락을 받고 공로패를 전달하다.

신안건토의

1) 서기가 보고한 2012년 회의록 자료 제9차 임원모임에서 상정한 회칙수정안을 보고한 자 료대로 받기로 박영실목사가 동의하고 김충현목사가 재청하여 회칙을 수정하다.

수정내용 : 4장 11조 (임원의자격) 본회가입 5년이상된 자를 삭제하기로 하다.

12조 3항을 삭제하기로 하다. (연임하는 임원은 본 임원회 활동에 70%이 상 참석 및 회비 완납의 의무를 이행하는 자라야 함)

2) 기타 잡무는 임원회에 일임하기로 하고 폐회에 동의 박영실목사 재청 김충현목사가 하고 회장이 기도하고 모든 회의를 마치니 11시 50분 이었더라.

제 1 차 임원모임

일 시 : 2012년 11월 29일 오후 3시 30분

장 소 : 고구마 북카페

모인수 : 6명

조시현목사,최명헌목사,박영실,이성진목사,류성원목사,박광희목사

최명헌목사(상임부회장)의 기도로 회의를 시작하다.

결의내용

1) 부회장 중 감리교단 부회장을 인선하다.

합동 : 박영실목사, 신동렬목사

감리 : 나진균목사

기타 : 권오란목사

2) 신년하례회는 지역교회의 목회자와, 지역기관,단체장까지 참석하여 진행하기로 하다.

일시 2013년 1월 9일(수) 오전 7시 장소는 미정.

사회는 상임부회장 / 설교는 회장 / 축도는 직전회장 / 대표기도는 부회장(박영실목사)

3) 신년축복성회는 1월 11일(금) 오후 8시 장소는 미정.

강사는 회장과 총무, 서기에게 일임하기로 하다.

4) 승리교회(지역내 군부대교회) 지원은 하기로 하다. 단, 임원단이 대대장을 만나서 전달 할 것을 추진하기로 하다.

5) 회계 통장을 만들기로 하다.

박영실목사 폐회 동의 후 총무(이성진목사)의 기도로 마치다.

제2차 임원모임

일 시 : 2013년 1월 4일 오전 8시 30분

장 소 : 조암감리교회

모인수 : 7명

조시현목사,최명헌목사,박영실,이성진목사,류성원목사,박광희목사,엄영주목사

이성진목사의 기도로 회의를 시작하다.

2. 신년축복성회 순서자를 정하다.

사회 / 조시현목사 대표기도 / 최명헌목사 성경봉독 / 류성원목사

헌금기도 / 박광희목사 광고 / 이성진목사 축도 / 엄영주목사

특별기도 순서 담당은 증경회장단 중 가능한 분들로 하기로 하며 이 일은 총무에게 일임 을 하기로 하다.

또한 신년축복성회에 많은 참석을 위하여 최선을 다하기로 하다.

신년하례 및 조찬기도회 결과

1. 행사일시 : 2013년 1월 9일(수) 오전 8시

2. 장소 : 조암감리교회

3. 참석인원 : 37명

목회자 14명

기관장 8명 (금종례 도의원. 홍사승 우정읍장, 이응구 장안면장, 남동학파출소장

김요한정보관. 임채준 3075부대장. 김강래 새마을금고이사장

김명구 서부신협이사장)

조암감리교회 장로들(5명)과 봉사자 (10명)

4. 오전 7시 예배 시작. 진행은 순서지 참조.

5. 광고전 3075부대에 지원금을 전달하다.

6. 조암감리교회 새로 부임한 나광호 목사의 인사가 있었다.

7. 예배와 하례 그리고 식사를 모두 마치니 8시 25분이었다.

신년축복성회 결과

1. 행사일시 : 2013년 1월 11일(금) 오후 8시

2. 장소 :화산장로교회

3. 참석인원 : 101명 (목회자 11명 포함)

4. 강사 : 박성민목사(한국대학생선교회대표)

5. 8시 예배를 시작하여 9시 30분에 마침. (진행은 순서지 참조)

제 3 차 임원모임

일 시 : 2013년 3월 13일 오전 9시

장 소 : 우정 식당

모인수 : 6명

조시현목사,최명헌목사,박영실,이성진목사,류성원목사,박광희목사

최명헌 목사 기도로 시작하다.

결의내용

1. 부활절 연합예배 순서자

사회 / 회장(조시현목사)

대표기도 / 상임부회장(최명헌목사)

성경봉독 / 서기(류성원목사)

설교 / 장순직목사(드림교회담임)

특별기도 / 최현재목사, 이명식목사, 김충현목사, 엄영주목사

기도순서는 증경회장순으로, 제목은 총무일임

헌금기도 / 회계(박광희목사)

광고 / 총무(이성진목사)

축도 / 나광호목사

예배 전 찬양인도, 성가찬양, 예배 안내위원 및 헌금위원, 간식은 조암감리교회에 부탁한 다.

2. 각 교회별 분담금은 광고 찬조 및 당일 헌금으로 대체한다.

3. 부활절 헌금 사용은 순수 경비를 지출 후 잔액을 가지고 추후 논의하기로 하다.

2. 장소 : 조암감리교회

3. 참석인원 : 280명 (목회자 19명 포함)

4. 강사 : 장순직목사(드림교회담임)

5. 오후 3시 예배를 시작하여 4시 40분에 마침. (진행은 순서지 참조)
